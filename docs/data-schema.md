# Data Schema

## `data/processed/draws.csv`

| Column | Meaning |
| --- | --- |
| `draw_date` | Draw date in ISO format. |
| `region` | Region identifier, initially `mien-bac`. |
| `source_name` | Public source identifier. |
| `source_url` | Public URL used for retrieval. |
| `collected_at` | UTC collection timestamp. |
| `validation_status` | Accepted, rejected, or warning status. |
| `validation_messages` | Semicolon-separated validation notes. |
| `content_hash` | Stable hash of normalized prize content. |

## `data/processed/prizes.csv`

| Column | Meaning |
| --- | --- |
| `draw_date` | Parent draw date. |
| `prize_group` | Canonical prize group. |
| `position` | Position within the prize group. |
| `raw_value` | Value extracted from source content. |
| `normalized_value` | Digits-only normalized value. |
| `last_two_digits` | Two-digit analytical number. |

## `data/reports/*.csv`

Metric outputs are regenerated by `vn-lottery analyze` and consumed by `vn-lottery report`, `vn-lottery visualize`, and `vn-lottery run-daily`.
