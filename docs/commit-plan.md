# Milestone Commit Plan

1. `foundation`: project metadata, package skeleton, settings, logging, domain models, CLI shell, and baseline tests.
2. `collection-storage`: cache, HTTP client, parser, normalizer, CSV store, incremental update, and collection tests.
3. `analytics`: frequency, rolling frequency, recency, monthly distribution, heatmap, histogram, moving average, and metric outputs.
4. `publishing`: static charts, dashboard, Markdown report, and GitHub Pages-ready docs.
5. `automation`: CI, scheduled update workflow, commit-if-changed behavior, and Pages deployment.
6. `release-readiness`: README, schema docs, maintenance guide, validation notes, linting, typing, and full test results.
